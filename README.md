# MadFab

## About
#### Webpage built for a baked goods company.

## Running MadFab locally
#### To run MadFab locally, run the following commands in your terminal:

```sh
git clone https://github.com/gabepages/MadFab.git
```

```sh
cd MadFab
```

```sh
npm install
```

```sh
npm run server
```
